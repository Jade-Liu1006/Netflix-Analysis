# Netflix Data Analysis

## Project Overview

In this project, I explored Netflix’s content dataset to understand how its content has changed over time. I focused on trends in content growth, country distribution, and genre patterns.

---

## What I Did

* Cleaned and processed the dataset using Pandas
* Analyzed how the number of titles changed over time
* Identified the top contributing countries
* Explored the most common content categories
* Visualized key trends using Matplotlib

---

## Key Findings

**1. Content Trend**
Netflix content started to increase significantly around 2015, reached a peak in 2019, and then slightly declined.

**2. Country Distribution**
The United States has the largest number of titles, followed by India and the United Kingdom.
This shows that Netflix still relies heavily on the U.S., but is also expanding into other markets.

**3. Genre Distribution**
International Movies are the most common category, while Romantic Movies appear less frequently.
This may suggest that Netflix focuses more on content that can reach a global audience.
